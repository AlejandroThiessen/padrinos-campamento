# Padrinos del Campamento Juvenil

Registro público y transparente de patrocinios para el torneo del campamento juvenil.
Meta: **$25,000 MXN**.

**Ver el tablero en vivo:** https://alejandrothiessen.github.io/padrinos-campamento/

Cada donativo queda registrado con folio, fecha y monto. Los patrocinadores anónimos
aparecen con un código privado que solo ellos conocen. Los gastos también se publican,
para que el dinero se pueda seguir de principio a fin.

*Para instrucciones de actualización (agentes/asistentes), ver `AGENTS.md`.*
